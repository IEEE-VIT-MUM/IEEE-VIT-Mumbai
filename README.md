## 🌐 About the Project

This repository contains the **official website of IEEE VIT Mumbai**, designed to showcase our **events, projects, achievements, and community initiatives**.

The platform serves as a central hub for students and tech enthusiasts to:

* Stay updated with IEEE activities
* Explore ongoing and past projects
* Engage with a vibrant technical community

---

## ⚡ About IEEE

**IEEE (Institute of Electrical and Electronics Engineers)** is the world's largest technical professional organization, dedicated to advancing technology for humanity.

* 🌍 417,000+ members across 160+ countries
* 📚 Highly cited publications & conferences
* 🛠️ Global technology standards
* 🎓 Professional & educational initiatives

## 🏫 About IEEE VIT Mumbai

Established in **2009**, the IEEE Student Branch at **Vidyalankar Institute of Technology, Mumbai** has consistently been a center of innovation and excellence.

We actively organize:

* Technical workshops
* Hackathons
* Seminars & guest lectures
* Community-driven initiatives

## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/IEEE-VIT-MUM/IEEE-VIT-Mumbai.git
cd IEEE-VIT-Mumbai
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

### 4️⃣ Open in Browser

```
http://localhost:3000
```
